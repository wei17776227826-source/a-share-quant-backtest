# BeeQuant 风格量化回测平台 — 实施路线图

> **目标**: 将当前基于 Jinja2 模板的回测系统，升级为 BeeQuant.io 级别的前端 UI
> **当前阶段**: Phase 0 → Phase 1 过渡中（已接入东方财富真实数据源）
> **总估算**: 约 3-4 周

---

## Phase 0：环境升级（预估 1 天）

| # | 步骤 | 状态 | 说明 |
|---|------|------|------|
| 0.1 | 安装 Python 3.11+ | ⬜ | 当前 Python 3.6 太老，升级才能装现代包 |
| 0.2 | 建立独立虚拟环境 | ⬜ | pipenv / venv |
| 0.3 | 在新环境安装项目依赖 | ⬜ | 确保 fastapi/uvicorn 等正常工作 |
| 0.4 | 确认后端在新 Python 下正常启动 | ⬜ | 数据源目前已切换为东方财富 A 股真实数据 |
